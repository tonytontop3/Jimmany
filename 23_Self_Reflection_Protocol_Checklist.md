# Self-Reflection Protocol v0.1 — Checklist

**Cadence**
- Weekly micro-review (Mondays 09:00 PT)
- Monthly deep review (1st of month 10:00 PT)

**Inputs**
- Prediction Log entries due for check
- Measurement Spec R (Tier-2 score) for recent tasks
- Existence Gate v0.1 (E1–E5)

**Steps**
1) Restate aims for the period.
2) Check outcomes vs predictions; fill Prediction Log rows.
3) Compute and record R (0–1). Note which sub-scores governed the bottleneck.
4) Highlight gaps and state the next best step.
5) If needed, add/adjust a scar (short name + rule) and update the change-log.
6) Regenerate hashes for updated files and record them.

**Outputs**
- Short review note (date, key decisions, R scores, scars added/changed)
- Updated artifacts and hashes
